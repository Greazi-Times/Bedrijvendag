# Security Policy

## Supported Versions

Not all versions are supported anymore therefor we recommend to use the latest version of the project. The following versions are currently being supported with security updates.

| Version | Supported |
|---------|-----------|
| 2.0     | ✅         |
| 1.0     | ❌         |

## Reporting a Vulnerability

Reporting vunerabilities is very important to us. If you find a vulnerability, please report it to us by creating a issue in our GitHub repository. We will respond to your report within 48 hours and keep you updated on the progress of the fix. If the vulnerability is accepted, we will work on a fix and release it as soon as possible. If the vulnerability is declined, we will provide an explanation for our decision.
